GSI-SSHTerm Applet (Kerberos)

Please note, you will require at least JDK 1.6 to launch the applet!
It has been tested with the following versions (all x86 architecture):
JDK6 for Windows
JDK6 for Linux
JDK6 for Solaris


Download GSI-SSHTerm-0.63k.zip as a standalone application

Note for Windows:

Please disable the caching of jar's by Java otherwise this will not run as an applet. Go to control panel and select Java. Click "Settings..." in the "Temporary Internet Files" section, in the window that opens select "Delete Files..."->"Ok" and then unselect "Keep temporary files on my computer". Click "Ok" on both dialog boxes and restart your browser.
