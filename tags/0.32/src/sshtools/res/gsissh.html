This applet is designed to allow you to easily and securely access the Grid using GSISSH. By default it should work with any UK e-Science Grid where you have an account (e.g. the NGS) using your UK e-Science certificate or a proxy certificate stored in the MyProxy server. For more details click here or read the FAQ.

Please note, you will require Java Software Development Kit (known as a JDK or SDK) 1.5 or higher installed to launch the applet!
